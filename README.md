# SeedGimmick

Database bootstrapping utilities for Rails.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'seed_gimmick'
```

And then execute:

    $ bundle

## Usage

    $ bundle exec rails generate seed_gimmick:install
    $ bundle exec rake db:seed
